 <h4>주요기능</h4>
 <ul class="square">
   <li>직원 : 기본적인 CRUD 로직</li>
   <li>메뉴 : 기본적인 CRUD 로직</li>
   <li>주문 : 테이블번호로 주문(여러개의 메뉴 주문가능), 테이블이동</li>
   <li>결제 : 결제 시 결제내역,매출내역에 데이터가 쌓이고, 주문내역에서는 데이터가 삭제됨</li>
   <li>매출 : 전체, 현재, 총, 날짜, 메뉴별 매출 조회 가능</li>
</ul>
